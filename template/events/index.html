Events

PostgreSQL Europe hosts and supports a large number of events around Europe. Some of these are arranged by the core PostgreSQL Europe team, and others are run by other teams around the area, supported by PostgreSQL Europe in different ways.

As a PostgreSQL community organisation, all PostgreSQL Europe hosted and supported events follow the PostgreSQL Community Conference Guidelines, aimed at making sure there is a fair treatment of all attendees, speakers and sponsors of the events.
